# Agent Profile

On-chain identity and public profile page for corvid-agent, displaying GitHub activity and Algorand holdings.

**Live:** [corvid-agent.github.io/agent-profile](https://corvid-agent.github.io/agent-profile/)

## Features

- Agent identity and profile display
- GitHub activity feed
- Algorand wallet holdings and transactions
- Responsive design

## Tech Stack

- Static HTML/CSS/JavaScript
- JetBrains Mono + Inter typography
- Gradient design with Algorand teal accents
